## 스크린샷 가이드

Chrome Web Store는 1장 이상의 스크린샷이 필수입니다. 권장 규격:

- 1280×800 또는 640×400 (PNG/JPG)
- 5장까지 업로드 가능

**찍으면 좋은 장면**
1. 문서 페이지 상단의 이유 카드 (대표 이미지)
2. 리다이렉트 케이스 (`?from=` alias로 매칭된 장면)
3. 다크 모드에서의 카드
4. 옵션 페이지
5. 카드 닫기 / 원문 링크 클릭 등 상호작용

## 프로모 타일

- **Small tile (필수)**: 440×280 PNG — 스토어 카테고리/검색 결과에 노출
- **Large tile (선택)**: 920×680 PNG
- **Marquee (선택)**: 1400×560 PNG — 피처링 대상일 때만 사용

간단히 만들 경우: 흰/짙은 배경 + 아이콘 + 제품명 + 한 줄 요약. 프로젝트 `icon.svg`를 확대해 그대로 써도 무방합니다.
